# SAC - Engine de Gestión Tributaria (API) ⚙️

Núcleo lógico del **Sistema de Administración Central (SAC)**. Maneja la arquitectura de datos, el cumplimiento regulatorio y la persistencia de información tributaria.

## 🛠️ Stack Tecnológico
- **Runtime:** Node.js + TypeScript
- **ORM:** Prisma
- **Base de Datos:** PostgreSQL
- **Cálculos:** Módulos optimizados en Python
